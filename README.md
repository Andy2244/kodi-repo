# Andy2244's Kodi Repository

A personal Kodi addon repository.

## Installation

1. Download: [repository.andy2244-1.0.0.zip](https://github.com/Andy2244/kodi-repo/raw/main/zips/repository.andy2244/repository.andy2244-1.0.0.zip)
2. In Kodi: **Settings** → **Add-ons** → **Install from zip file**
3. Select the downloaded zip
4. Go to **Install from repository** → **Andy2244's Repository**

> **CoreELEC/LibreELEC:** If installation fails, go to **Power** → **Restart Kodi** and try again.

## Available Addons

| Addon | Description |
|-------|-------------|
