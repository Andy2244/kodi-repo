# Andy2244's Kodi Repository

A personal Kodi addon repository.

## Installation

### Option 1: Add Source URL (Recommended)

No download required:

1. In Kodi: **Settings** → **File Manager** → **Add source**
2. Click `<None>` and enter: `https://raw.githubusercontent.com/Andy2244/kodi-repo/main/zips/`
3. Name it `Andy2244` and click **OK**
4. Go to **Settings** → **Add-ons** → **Install from zip file**
5. Select `Andy2244` → `repository.andy2244` → install the zip
6. Done! Now install addons from **Install from repository** → **Andy2244's Repository**

### Option 2: Manual ZIP Download

1. Download: [repository.andy2244-1.0.1.zip](https://github.com/Andy2244/kodi-repo/raw/main/zips/repository.andy2244/repository.andy2244-1.0.1.zip)
2. In Kodi: **Settings** → **Add-ons** → **Install from zip file**
3. Select the downloaded zip

> **CoreELEC/LibreELEC:** If installation fails, go to **Power** → **Restart Kodi** and try again.
